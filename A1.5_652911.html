Gener una matriz de correlacin y la represent grficamente con un heatmap utilizando seaborn. La correlacin mide la fuerza y direccin de la relacin lineal entre dos variables numricas, con valores que van de -1 a 1. Cercano a 1: fuerte correlacion positiva(cuando una variable aumenta, laa otra tambin). Cercano a -1: fuerte correlacin negativa (cuando una variable aumenta, la otra tiende a disminuir). Cercano a 0: poco o nula relacin lineal.

Este analisis es importante porque la regresin lineal mltiple supone que los predictores no estan fuertemente correlacionados entre si. Cuando dos o mas vairables indpendientes presentan correlaciones muy altas, se produce el problema de multicolinealidad, que tiene varias consecuencias negativas, como puede llevar a que el modelo sea inestable, pequeos cambios en los datos generan grandes cambios en los coeficientes. En este conjuno de datos era esperable que las calificaciones G1 y G2 estn muy correlacionadas con G3(calificacion final), ya que son mediciones del mismo rendimiento acadmico en distintos periodos. Tambin es imporante revisar si G1 y G2 estan correlacionadas entre si.

El heatmap ayuda a la deteccin de estos patrones de correlacin porque utiliza una esscala de colores, lo que permite identificar rpidamente qu variables presentan posibles problemas de colinealidad.

import matplotlib.pyplot as plt
import statsmodels.api as sm
from sklearn.model_selection import train_test_split

x = df_mod.drop(columns=["G3"]).apply(pd.to_numeric, errors="coerce")
y = pd.to_numeric(df_mod["G3"], errors="coerce")

x_train, x_test, y_train, y_test = train_test_split(
    x, y, test_size=0.2, random_state=55
)

x_train_const = sm.add_constant(x_train, has_constant="add")
model = sm.OLS(y_train.astype(float), x_train_const.astype(float)).fit()
print(model.summary())

x_test_const = sm.add_constant(x_test, has_constant="add")
y_pred = model.predict(x_test_const.astype(float))

print("Tamao y_test:", len(y_test), " | Tamao y_pred:", len(y_pred))

plt.figure(figsize=(6,6))
plt.scatter(y_test, y_pred, alpha=0.7)
mn, mx = y.min(), y.max()
plt.plot([mn, mx], [mn, mx], linestyle="--") 
plt.xlabel("Calificacin real G3")
plt.ylabel("Calificacin predicha G3")
plt.title("Valores reales vs. predichos (OLS)")
plt.show()

print("Conclusin: mientras ms se acerquen los puntos a la lnea discontinua, mejor el desempeo del modelo.")
